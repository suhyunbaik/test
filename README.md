## Deploy
### How does it work  

### Steps
1. create gh-pages branch  
```git checkout -b gh-pages```

2. git push to create branch remote  
```git push origin gh-pages```

3. run deploy script  
```npm run-script deploy```

4. delete branch local and remote  
```git branch -D gh-pages```
